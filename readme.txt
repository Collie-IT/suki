== Description ==

Suki is a free multi-purpose WordPress theme that is lightweight, fast, highly customizable, and flexible. Build any kind of websites, blog, business, portfolio, WooCommerce storefronts, and also niche websites like Spa, Restaurant, Wedding, Non Profit, Education, Hotel, etc. Fully compatible with Gutenberg, Elementor, Brizy, Visual Composer, Beaver Builder, Divi, SiteOrigin, and other page builders. Suki includes complete design options for typography, colors, layout, and also free built-in Header and Footer builder. Built with SEO in mind, Suki is incredibly fast, mobile optimized, 100% HTML valid, and integrated with Schema.org structured data. Suki also supports multi-language translation and RTL ready. Learn more about Suki features and our one click demo import feature at https://sukiwp.com/.

== Installation ==

1. Go to Appearance > Themes and click the Add New button.
2. Search for "Suki" and wait until the theme appears.
3. Click Install Now button and then Activate the theme right away.
